# Git and GitHub Fundamentals | WoMakersCode

This repository contains the notes, practical exercises, and documentation developed during the Git and GitHub module of the WoMakersCode Data Analytics Bootcamp. 

The course, instructed by [Kamila Santos](https://github.com/Kamilahsantos), focused on industry-standard version control workflows and collaborative development.

---

## Curriculum Overview

The training was structured into six comprehensive modules:

| Module | Core Topics |
| ------ | ----------- |
| Module 1 | Initial concepts, version control systems, and environment setup. |
| Module 2 | Repository initialization and essential commands (add, commit, push, pull). |
| Module 3 | Branching strategies, merging, and conflict resolution. |
| Module 4 | Open Source collaboration, Forking, and Pull Requests. |
| Module 5 | Project management using GitHub Projects and issue tracking. |
| Module 6 | Advanced tips, best practices, and final review. |

---

## Key Skills Acquired

* **Version Control:** Managing project history and maintaining code integrity.
* **Collaboration:** Working on multi-developer environments via Pull Requests and code reviews.
* **Project Management:** Organizing tasks and workflows using GitHub's native tools.
